# nore-landing

Public NORE landing/funnel pages.

## Pages

- `/` role selector
- `/trainer` trainer landing page
- `/operator` operator landing page
- `/trainer/try` trainer static demo
- `/operator/try` operator static demo
- `/checkout` public checkout handoff page

## Local preview

```bash
python3 -m http.server 4173
```

Open:

- http://localhost:4173/
- http://localhost:4173/trainer/
- http://localhost:4173/operator/
- http://localhost:4173/trainer/try/
- http://localhost:4173/operator/try/
- http://localhost:4173/checkout/
